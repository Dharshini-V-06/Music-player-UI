# Music Player UI (HTML & CSS Only)

A simple and elegant **Music Player User Interface** built using **pure HTML and CSS**.
This project demonstrates how interactive UI elements and animations can be created **without JavaScript**.

---

## Features

* Minimal and modern music player design
* Play / Pause toggle using CSS checkbox hack
* Animated progress bar (CSS only)
* Hover effects on album art and player card
* Gradient background with smooth transitions
* No JavaScript, no external libraries

---

## Technologies Used

* **HTML5**
* **CSS3**
